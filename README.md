# Launch-Page
🚀 Animated Landing Page (HTML & CSS)

A simple landing page built using HTML and CSS to showcase CSS animations and transitions.

🚀 Features

🎬 Smooth CSS animations

✨ Animated elements on load

🎨 Modern UI with gradients

📱 Basic responsive layout

⚡ No JavaScript (pure CSS animation)

🧠 What I Learned

Using @keyframes for animations

Applying transition and transform

Controlling animation timing and delays

Improving UI with motion and effects

Structuring a basic landing page

🛠️ Tech Stack

🌐 HTML

🎨 CSS (Animations + Transitions)
